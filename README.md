# Citi_bike-analysis
An analysis using open source citi bike data from 2022

# Key Tasks Performed
- Cleaned and wrangled data efficiently
- Merged data frames
- Derived new variables and columns from existing tables
- Documented changes and used markdowns to explain my process
- Created a tableau dashboard to effectively display insights and show what areas we can explore further
- Created a presentation simplifying and exploring the insights discovered
- Performed aggregate functions in combination with filtering, sorting, and grouping data
- Created flags based on rental and trip information
- Performed exploratory analysis to obtain additional insights

# Procedures
- Correlation Matrix Heat Map

- Regression Analysis
  - Creating Training and Test sets
  - Regression Line Scatterplot

- Time Series Analysis

  - Differencing
